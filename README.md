# GÉNOVA SRL - ORGANIZER 🏢

Sistema de gestión centralizada y administración interna para edificios y propiedades de **GÉNOVA SRL**. El objetivo principal de esta aplicación es resolver problemas recurrentes de organización, flujos de pago y registro de eventos diarios de manera rápida, fluida y clara.

## 🚀 Características Principales (Fase 1)

1. **Gestión de Facturas y Presupuestos (Dashboard):**
   * Registro manual de facturas (Número, monto, proveedor, edificio asignado).
   * Control de estados de pago (Pendiente, pago parcial con cálculo de porcentaje automático, abonado total).
   * Sistema de recordatorios de saldos y fechas de vencimiento.
   * Registro de aprobación (Quién aprobó y soporte para adjuntar capturas/vínculos de aprobación).

2. **Centralización de Información de Edificios:**
   * Directorio de proveedores, services, teléfonos, mails y métodos de pago de cada uno.
   * Información de cuentas bancarias por edificio.
   * Registro de personal (Funcionarios directos o empresas de limpieza/vigilancia), incluyendo horarios y días laborales.

3. **Bitácora Digital:**
   * Historial de informes diarios por edificio para mantener un registro de novedades, incidencias o eventos relevantes día a día.

---

## 🛠️ Stack Tecnológico Proyectado

* **Frontend:** ReactJS + Vite (Para un entorno de desarrollo ultra rápido y modular).
* **Estilos:** Tailwind CSS (Diseño ágil enfocado en la usabilidad y orden visual).
* **Backend & Base de Datos:** Supabase (PostgreSQL relacional para vincular edificios, facturas y proveedores de forma consistente).
* **Autenticación:** Supabase Auth (Control de acceso seguro para usuarios autorizados).
